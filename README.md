## Практика по ручному тестированию на курсе QA инженер от Avito

### Разработка тест-кейсов и чек-листа

В ходе выполнения работы удалось проанализировать требования и предоставленный дизайн на платформе figma для тестового web-приложения TestBoard. Функционал площадки представлял собой доску объявлений. Результатом работы стал набор тест-кейсов и чек-лист по проверке основной функциональности платформы. Также, в ходе проверки приложения по составленной документации, удалось составить несколько баг-репортов на площадке Яндекс Трекер.
